SynodicRhythm.php : 19

A collection of Moon SynodicRhythmRecord instances.

Represents the Moon's synodic rhythm over a time range. A Moon synodic rhythm, or synodic period, is the time it takes for a celestial object to return to the same position relative to the Sun, as seen from the Earth.
